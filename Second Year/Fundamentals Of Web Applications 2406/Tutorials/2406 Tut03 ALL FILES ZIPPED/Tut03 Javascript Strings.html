Problem 1)

Download the demo_code folder and examine the javascript in the file problem1.js. It should look like that shown below. 

function landscape() {

  let result = ""

  function flat(size) {
    for (let count = 0; count < size; count++)
      result += "_"
  }

  function hill(size) {
    result += "/"
    for (let count = 0; count < size; count++)
      result += "'"
    result += "\\"
  }

  //BUILD SCRIPT
  flat(3)
  hill(4)
  flat(6)
  hill(1)
  flat(1)
  //END SCRIPT

  return result

}

console.log("")
console.log(landscape())

Run the code with node.js. It will write output to the console window (You don't need the browser.) It should produce a simple string output meant to look like a landscape profile like this:

___/''''\______/'\_


For problem 1 modify this code so that it uses the '_' underscore character for both the flat portions and the tops of the hills. That is, so that the following build script portion:

//BUILD SCRIPT
flat(3)
hill(5)
flat(2)
hill(3)
flat(4)
hill(0)
flat(2)
//END SCRIPT

produces output that looks like this:

 
      _____    ___
  ___/     \__/   \____/\_

To do this the underscore characters that form the top of the hills must be printed on the previous line of console output. That is, the output represents two lines of text unlike that of the demo code. Also notice that the hill can have a top with no flat portions so the argument passed to the hill() function represents width of the hill excluding its '/' and '\' characters.

To do this exercise you can use whatever you want from javascript: strings, arrays, objects whatever, though the solution does not require anything more complex than strings. You will have to figure out how to use the features you choose in javascript by reading about the language.

Study the code example. Can you explain the output. While it illustrates how to colour the text it shows something much more disturbing: it is unsafe to use a FOR-IN loop to loop over the characters of javascript strings. You should always use a for(let i=0; i<x.length; i++) loop to go through the characters of a string, or the elements of an array (Javacript ES5). Again, be warned: it is unsafe for use a FOR-IN loop to traverse the characters of a string or the elements of an array in javascript.

[ES6] More recently in javascript ES6 (also known as ECMAScript6 and ECAMScript2015) they have added a new FOR-OF loop to deal with this fiasco. It IS SAFE to loop over the characters of a string or elements of an array using a FOR-OF loop like the following (try it in your code for this problem.)

let str = "Hello World"
for(let ch of str) console.log(ch)
