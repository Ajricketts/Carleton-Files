Program Author: Alyxander-Jacob Ricketts

Purpose: A conversion Funciton that allows the user to convert an integer from base 10 to a number in another base.

List of Source Files: convert, convert.c, convert.h, main.c, README.txt

Compilation Command: gcc main.c convert.c

Launching and operating instructions: Compile the code using the compilation command above in terminal. Using the program put the number that is intended to be converted as the first variable and the base you wish to convert it to as the second variable.
